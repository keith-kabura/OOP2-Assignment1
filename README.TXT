OOP II Assignment 1

Description
This project is a Java-based Car Rental System developed using Object-Oriented Programming principles.
It also includes a login system with password validation and limited login attempts.

Technologies Used
- Java
- BlueJ IDE

 Features
- Car rental and return
- Customer management
- Login system with 3 attempts
- Password masking

How to Run
1. Open the project in BlueJ
2. Compile all classes
3. Run LoginSystem.java
4. Run Main.java for the Car Rental System
